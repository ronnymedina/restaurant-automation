Problema de routing — /kiosk/[slug] en modo estático

Problema: Astro static necesita conocer todos los slugs en build time para generar /kiosk/mi-restaurante/index.html. Los slugs vienen de la base de datos — imposible en build time.

Solución: Renombrar la página a src/pages/kiosk/index.astro y configurar NestJS para que cualquier request a /kiosk/* sirva ese HTML. El slug se lee desde window.location.pathname en el React island.

// KioskApp.tsx — leer slug del path
const slug = window.location.pathname.split('/kiosk/')[1]

// NestJS — wildcard handler para SPA
@Get('/kiosk/*')
serveKiosk(@Res() res) {
  res.sendFile('kiosk/index.html', { root: 'public' })
}
