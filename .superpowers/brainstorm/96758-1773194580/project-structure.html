Pipeline de Build

1
Build NestJS
pnpm --filter api-core build
Compila TypeScript → dist/
2
Build Astro (estático)
pnpm --filter ui-dashboard build
Genera HTML/JS/CSS en dist/
3
Obfuscación del JS
javascript-obfuscator dist/
Ofusca el output compilado de NestJS antes de empaquetar
4
Electron Builder
electron-builder --win --mac
Empaqueta todo + firma el binario
Windows: NSIS installer (.exe) macOS: .dmg + notarización Apple
5
Publicar release
GitHub Releases / S3
electron-updater descarga actualizaciones desde aquí automáticamente
Auto-update incluido
